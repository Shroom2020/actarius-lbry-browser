Actarius LBRY Browser: Release 0.2

Actarius is an Electron-based browser that can view LBRY URLs. Currently, the LBRY network is dominated by LBRY Inc developed apps, Odysee and the LBRY Desktop, as the main way to access content. There are third-party solutions, like the terminal based FastLBRY project. We find this to be unintuitive, insufficient, and/or too centralized.
Our goal is to develop a user-friendly, open-source LBRY network browser that communicates directly with the LBRY network and has minimal dependency on any other entity.
